Links para download

  • Baixar PDF
  • Baixar arquivo .RAR

Se liga em como ficou o código pra esses download aqui de cima:


    <ul>
        <li><a href="livro/10 - Ligações em toda parte.pdf" target="_blank" download="10 - Ligações em toda parte.pdf" type="application/pdf">Baixar PDF</a></li>
        <li><a href="livro/10 - Ligações em toda parte.rar" download="10 - Ligações em toda parte.pdf" type="application/x-rar-compressed">Baixar arquivo .RAR</a></li>
    </ul>

O parâmetro download será o nome do arquivo a ser baixado
O type indica ao navegador o tipo do arquivo a ser baixado
Uma lista de media types (tbm conhecidos como MIME) pode ser encontrada em:

  • iana.org/assignments/media-types
  • E tem esse do mozilla que eu achei o .rar (application/x-rar-compressed)
